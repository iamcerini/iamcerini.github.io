About PsychSpace

Psychspace encourages users to understand the science behind self-motivation, turning ambitions into accomplishments, correcting bad habits, and maintaining new routines. Users can sign up as students and select courses that would help them create or correct specific habits. As part of an effort to promote cyberlearning, PsychSpace helps students manage their self-enrolled topics, keep track of their progress during the learning period, communicate with peers, and receive personalized feedback from instructors proficient in psychology and self-help techniques.

My Responsibilities

As Lead Designer, I was responsible for the look, feel, and interactivity of the web app. I coded and managed the frontend JSP, CSS, and Javascript files for the student users and created the graphics featured on the site. The logo was created using Adobe Illustrator and the stock photos were edited in Adobe Photoshop.

Technologies & Frameworks

PsychSpace applies Google AppEngine frameworks and RESTful web services. Backend was developed in Java, SpringMVC, and Google Datastore, and frontend was primarily developed in Javascript, JSTL, HTML, and CSS. Javascript technologies and libraries used include Bootstrap and jQuery. Any graphics and artwork were developed using Adobe Photoshop and Illustrator.

Version Control & Project Management

My team used GitHub to manage our application. The repository can be accessed here. To keep track of our group assignments and individual responsibilities, we used the project management tool Asana.

Research & Development

Since a large part of project was Psychology based, we interviewed nine psychology professors of Stony Brook University in both Cognitive Psychology and Social and Health Psychology so that we could develop a better understanding of the science behind habit formation and improve our application.

Written Documentation

Before developing the web app, a RFP for the National Science Foundation's Cyberlearning and Future Learning Technologies Grant as well as a Software Requirements Specification (SRS) using the IEEE 830-1998 standard were written.

About ePoGee

ePoGee is an application for creating an eportfolio or a website that summarizes one’s body of work.

Technologies & Development

This application was developed using JavaFX and CSS, utilizing the MVC design structure. A single JSON file was used to save and organize each project's data.

Customization Features

With hundreds of unique combinations, users can choose between 5 color themes, 5 fonts, either a vertical or horizontal page flow, a banner image, and other ways to align and customize a website. The user can also add an unlimited amount of pages to their website and each page can have an unlimited amount of modules including text boxes, lists, single images, slideshows, and videos.

Program Features

The application features the ability to create, open, save, export, and preview a project. The export feature produces a fully functional single page website with all the HTML, CSS, Javascript, and user content in an organized directory structure that can be put directly onto a server.

About Lights On

Lights On is survival horror video game developed for Stony Brook University's Advanced Game Programming course during the fall semester of 2016. The game centers around a fictional underground laboratory where the player must survive a series of conditioning experiments and manipulate their environment using colored lights. Part of a three person team, my roles included Designer, Writer, Artist, Composer and Assistant Programmer.

Specifications

  • Engine - Unity3D

  • 3D Modeling - Blender

  • Platform - Microsoft Windows

  • Genre - Survival Horror

  • Mode - Single-player

Gameplay

Lights On is first-person game that blends the genres of survival horror, psychological horror, and adventure. Throughout the game, the player must find clues and modifications for their flashlight to solve various puzzles and progress through the levels.

The enemies in this game cannot be killed but their behavior can be changed by using different colored lights. Green light attracts the enemy to a specified location, blue light causes the enemy to be idle and red light triggers the enemy to seek out and attack the player. Most of the colored lights have been designed into the environment, but as the game progresses the player receives special modifications to produce these colors and solve more complex puzzles. The flashlight is also used to interact with the environment such as unlocking doors and activating switches.

Using the flashlight will reduce the flashlight’s charge, indicated in the HUD. Batteries can help restore the charge, however the number of batteries are limited and can sometimes make the player more noticeable to enemies. The player must balance the amount of time they use the flashlight and plan accordingly when solving puzzles. Charge is usually fully restored once the player completes an objective or progresses the game's story. It may also be restored by being attacked by the enemy, but the player and any progress made will be reset to the beginning of the level or a checkpoint if reached.

The game is designed to balance stealth with confrontation, creating scenarios where the player may first have to avoid the enemies and then directly manipulate their behavior. The player is also given opportunities to find and read notes hidden throughout the levels, which provide clues to solving the puzzles as well as plot exposition.

Development

Development for Lights On began in August of 2016, using the Unity3D engine with C# to develop the game and Blender to model and render the 3D graphics.

Many aspects of behavioral psychology such as color psychology, classical conditioning, hypnosis and thought reform were incorporated into the plot and gameplay during development.

In December of 2016, a demo consisting of three playable levels was presented for review and received positive reviews.

About Python for Beginners

While working at KidOYO, I designed course covers for the beginner Python courses to encourage KidOYO users to learn Python while conveying what the lesson goals.

Python?

Python is a programming language and is often used to introduce the basics of programming. Learning a new language, especially a programming language, can be a bit daunting, especially when its logo features two python snakes. When beginning a new programming language, it is important not get overwhelmed with the fear of failure or too much information. To familiarize the beginners to the goal of the lesson as well as the logo, the two pythons were given friendly and adventurous personalities that beginners would find approachable and engaging.

"Hello World!"

The first step to learning any new programming language is learning to make a simple program with the single purpose of displaying "Hello World" to the programmer. This is how the programmer knows that their environment is working properly.


Guess-the-Number, Hang-Man, and Choose-Your-Adventure

Having beginners learn to create games that are simple and well-known is a great way for them to familiarize themselves with the language. Plus, what is cooler than playing a game you made yourself?


Python and Firebase

While an important part of programming is learning languages, there are so many external libraries and platforms, such as Firebase, that are essential to creating and maintaining computer, web, and phone applications.
